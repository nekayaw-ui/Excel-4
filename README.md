Corporate Financial Analysis – Capital Budgeting & Investment Decision Project
Project Overview
This project evaluates two competing capital investment opportunities using industry-standard capital budgeting techniques commonly performed by Financial Analysts, FP&A Analysts, Corporate Finance Analysts, and Investment Analysts.
The objective was to determine which project would create the greatest shareholder value while understanding how changes in the firm's cost of capital influence investment decisions.
Rather than relying on a single financial metric, this analysis combines multiple valuation techniques to compare profitability, investment efficiency, project lifespan, and sensitivity to changing market conditions.
Business Scenario
A company must decide between two mutually exclusive investment projects.
Each project requires a different initial investment, generates unique future cash flows, and has different project lives.
The goal is to determine which project should be accepted using several corporate finance valuation techniques instead of relying on a single metric.
Step 1 – Organized Project Cash Flows
The first step involved organizing each project's projected cash inflows and initial investment into Microsoft Excel.
This established the financial model that served as the foundation for every subsequent analysis.
The cash flow schedule included:
Initial investment
Annual cash inflows
Project lifespan
Cost of capital assumptions
This financial model allows valuation techniques to calculate investment performance consistently across both projects.
Step 2 – Net Present Value (NPV) Analysis
Net Present Value (NPV) was calculated for both projects to determine the amount of value each investment adds to the company after considering the time value of money.
The analysis discounted every future cash flow back to today's dollars using the firm's required rate of return.
Key objectives included:
Measuring shareholder value creation
Comparing investment profitability
Identifying the project with the greatest economic value
The project with the higher positive NPV represents the investment expected to maximize firm value.
Step 3 – Internal Rate of Return (IRR)
The Internal Rate of Return (IRR) was calculated to determine each project's expected annual percentage return.
IRR identifies the discount rate where NPV equals zero and provides another perspective for comparing investment attractiveness.
The results were then compared against the firm's required rate of return to determine investment acceptability.
Step 4 – Modified Internal Rate of Return (MIRR)
Because traditional IRR assumes interim cash flows are reinvested at the IRR itself, Modified Internal Rate of Return (MIRR) was calculated to produce a more realistic estimate.
MIRR improves investment analysis by assuming cash flows are reinvested at the company's cost of capital rather than the project's IRR.
This provides a more practical measure of project profitability.
Step 5 – Discount Rate Sensitivity Analysis
A sensitivity analysis was performed by recalculating each project's Net Present Value across multiple discount rates.
This analysis demonstrates how changes in financing costs or market conditions affect investment profitability.
The exercise illustrates:
Interest rate risk
Cost of capital sensitivity
Stability of each investment under changing economic conditions
An NPV profile was generated to visualize how project value changes as discount rates increase.
Step 6 – Crossover Rate Analysis
The crossover rate was identified by comparing the NPV profiles of both projects.
This represents the discount rate where both investments generate identical Net Present Values.
Understanding the crossover point helps determine which project becomes more attractive as financing costs change.
This technique is commonly used when comparing mutually exclusive capital projects.
Step 7 – Equivalent Annual Annuity (EAA)
Since both investments have different project lives, Equivalent Annual Annuity (EAA) analysis was performed.
EAA converts each project's Net Present Value into an equivalent annual cash flow, allowing an apples-to-apples comparison.
This method is especially useful when deciding between projects with unequal operating periods.
Step 8 – Investment Recommendation
After comparing:
Net Present Value (NPV)
Internal Rate of Return (IRR)
Modified Internal Rate of Return (MIRR)
Discount Rate Sensitivity
Crossover Rate
Equivalent Annual Annuity (EAA)
a final investment recommendation was made based on the metric most appropriate for mutually exclusive projects.
The recommendation was supported using quantitative financial evidence rather than intuition.

Key Takeaways
This project demonstrates how multiple capital budgeting techniques work together to support strategic investment decisions. Rather than relying on a single financial metric, the analysis evaluates each project's value, expected return, risk under changing discount rates, and lifespan to recommend the investment that best maximizes shareholder value. The project reflects the type of financial modeling and investment evaluation commonly performed in corporate finance, FP&A, and financial analyst roles.
