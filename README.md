Corporate Financial Analysis – Capital Budgeting Project.

This project analyzes two investment opportunities using financial modeling techniques commonly used in corporate finance and financial analyst roles.

Key Analysis Performed:
Net Present Value (NPV),
Internal Rate of Return (IRR),
Modified Internal Rate of Return (MIRR),
Discount Rate Sensitivity Analysis,
Equivalent Annual Annuity (EAA).

Skills Demonstrated:
Financial Modeling,
Capital Budgeting,
Investment Analysis,
Discounted Cash Flow (DCF) Analysis,
Microsoft Excel.

Tools Used:
Microsoft Excel,
Financial Functions (NPV, IRR, MIRR),
Sensitivity Analysis.

Outcome:
Evaluated competing investment projects and applied data-driven financial decision-making techniques to identify the most value-creating opportunity.
